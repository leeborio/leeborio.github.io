Pandas Data Frame

In this post I’m going to present an example of how to create a pandas data frame “on the fly” from a Python dictionary

import pandas as pd
dixio = {
    'size': {1: 'XS', 2: 'S'}
    , 'color': {1: 'Green', 2: 'Red'}
    , 'status': {1: 'Active', 2: 'Inactive'}
}

DF = pd.DataFrame(dixio); DF

Output
size color status
1 XS Green Active
2 S Red Inactive

In the dictionary structure described above we are “embedding” the row index within the data declaration sections of the dictionary

Another way to construct a data frame from a dictionary is to “attach” the index to the dictionary when invoking the DataFrame function, as shown below

another_df = pd.DataFrame({
    'size': ['L', 'XL']
    , 'color': ['Blue', 'Yellow']
    , 'active': [True, False]
    }
    , index = [0, 1])

another_df

Output
size color active
0 L Blue True
1 XL Yellow False
